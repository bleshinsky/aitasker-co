# Positioning Scoring Rubric

Rate each of the 6 positioning components on a 0-10 scale. Use this rubric
to evaluate existing positioning or to quality-check new positioning output.

---

## Scoring Scale

| Score | Label | Description |
|-------|-------|-------------|
| 0-2 | **Missing** | Component is absent or completely undeveloped |
| 3-4 | **Weak** | Component exists but is vague, generic, or unsubstantiated |
| 5-6 | **Adequate** | Component is defined and directionally correct but lacks specificity or evidence |
| 7-8 | **Strong** | Component is specific, evidence-backed, and actionable |
| 9-10 | **Excellent** | Component is specific, evidence-backed, validated with customers, and clearly differentiated |

---

## Component Rubrics

### 1. Competitive Alternatives

| Score | Criteria |
|-------|----------|
| 0-2 | No competitive alternatives identified, or only "we have no competitors" |
| 3-4 | Lists a few direct competitors by name but does not include non-obvious alternatives (manual workarounds, hiring, status quo). Based on internal opinion, not customer data |
| 5-6 | Includes 3-5 alternatives spanning at least 2 types (direct + one other). Some customer input but not systematic |
| 7-8 | Includes 5+ alternatives spanning 3+ types. Ranked by frequency of customer mention. Each has documented strengths and weaknesses |
| 9-10 | All of 7-8, plus validated through customer interviews (5+), includes the primary "do nothing" alternative, and documents why customers leave each alternative |

### 2. Unique Attributes

| Score | Criteria |
|-------|----------|
| 0-2 | No unique attributes identified, or lists features that competitors also have |
| 3-4 | Lists features as differentiators but has not tested for uniqueness. "Better" is confused with "unique" |
| 5-6 | 3-5 attributes identified that are plausibly unique. Passes the Uniqueness Test for most. Not yet clustered |
| 7-8 | 3-5 clusters of unique attributes. Each passes Uniqueness Test and Truth Test. Evidence documented for each |
| 9-10 | All of 7-8, plus attributes are validated by customers ("this is why I chose you"), and durability is assessed (how long before competitors replicate) |

### 3. Value Mapping

| Score | Criteria |
|-------|----------|
| 0-2 | No value articulation, or value is expressed as features ("we have AI") |
| 3-4 | Value is stated as benefits but not connected to specific unique attributes. No "So What?" chain. Generic value statements |
| 5-6 | 2-3 value themes with "So What?" chains. Chains reach business outcomes but proof points are weak or missing |
| 7-8 | 2-4 value themes with complete "So What?" chains. Each has at least one proof point (customer quote, metric, or case study). Mapped to SaaS value categories |
| 9-10 | All of 7-8, plus value themes are validated through customer interviews, mapped to specific stakeholders, and quantified with metrics (before/after) |

### 4. Best-Fit Target Customers

| Score | Criteria |
|-------|----------|
| 0-2 | No target defined, or "everyone who needs [broad category]" |
| 3-4 | Target described in demographic terms only (e.g., "mid-market companies"). No behavioral or trigger-based criteria. Cannot pass the List Test |
| 5-6 | ICP defined with firmographic criteria. Buyer persona sketched. Can likely pass the List Test but has not been tested |
| 7-8 | ICP with firmographic + behavioral + trigger criteria. Buyer persona with role, pain, and buying behavior. Anti-persona defined. Passes the List Test |
| 9-10 | All of 7-8, plus validated with data (best customers match the profile), passes all three tests (List, Sales, Ad), and includes segment size estimate |

### 5. Market Category

| Score | Criteria |
|-------|----------|
| 0-2 | No category defined, or category is so broad it is meaningless ("software platform") |
| 3-4 | Category named but unclear strategy (Head-to-Head vs. Subcategory vs. New). Category fails the spoken test (people say "what does that mean?") |
| 5-6 | Category named with clear strategy chosen. Passes the spoken test. Adjacent categories considered. No education tax analysis |
| 7-8 | All of 5-6, plus education tax assessed, pricing alignment checked, and category name follows a recognized SaaS naming pattern |
| 9-10 | All of 7-8, plus category is validated with target customers (they naturally use this category or immediately understand it), and competitive framing within the category is clear |

### 6. Relevant Trends

| Score | Criteria |
|-------|----------|
| 0-2 | No trends identified, or generic trends with no connection to unique value ("AI is changing everything") |
| 3-4 | 1-2 trends identified but connection to unique value is forced or superficial |
| 5-6 | 1-2 trends with legitimate connection to unique attributes or value themes |
| 7-8 | 2-3 trends, each with a clear, specific connection to a unique attribute or value theme. Explains "why now" convincingly |
| 9-10 | All of 7-8, plus trends are supported by data (market research, analyst reports), and the connection is validated by customers ("yes, this is exactly why we looked for a solution now") |

## Overall Positioning Strength

Calculate the weighted average across all 6 components:

| Component | Weight | Rationale |
|-----------|--------|-----------|
| Competitive Alternatives | 15% | Foundation — everything else builds on this |
| Unique Attributes | 20% | The core of differentiation |
| Value Mapping | 25% | The most impactful component — value wins deals |
| Best-Fit Customers | 20% | Wrong target = wrong positioning, no matter how good the rest is |
| Market Category | 15% | Important but can be refined iteratively |
| Relevant Trends | 5% | Nice to have, not essential for initial positioning |

**Overall score interpretation:**

| Score | Assessment | Recommended Action |
|-------|-----------|-------------------|
| 0-3.0 | **Critical** — Positioning is absent or fundamentally broken | Full positioning build from scratch |
| 3.1-5.0 | **Weak** — Has a foundation but major gaps | Rebuild the 2-3 weakest components |
| 5.1-7.0 | **Adequate** — Directionally correct but lacks specificity or evidence | Sharpen weak components, add proof points, validate with customers |
| 7.1-8.5 | **Strong** — Solid positioning that can drive execution | Focus on validation and the 30/60/90 day plan |
| 8.6-10 | **Excellent** — Well-researched, validated, differentiated positioning | Maintain, validate quarterly, evolve as market shifts |
